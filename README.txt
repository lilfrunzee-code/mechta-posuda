# МЕЧТА — лендинг
Готовый одностраничный сайт + страница товара.

## Быстрый старт
Откройте `index.html` в браузере. Для хостинга — загрузите папку на любой статический хостинг (Netlify, Vercel, GitHub Pages, Nginx/Apache).

## Замены
- Телефон: `+7 843 000-00-00` в `index.html` и `product.html`
- E-mail: `partner@mechta.ru`
- Логотип: `/assets/logo-mechta-user.png` — замените на SVG/PNG.

## Формы
Вверху `index.html` в блоке Контакты уже есть разметка для Netlify Forms (`data-netlify="true"`). 
Если используете Formspree, замените форму на:
<form action="https://formspree.io/f/ВАШ_ID" method="POST"> ... </form>

## Маркетплейсы
Замените ссылки в секциях «Где купить» на адреса ваших карточек товаров.



## Цвета коллекции (ваши фото)
Замените в папке `/assets` файлы:
- `prod-blue.jpg` — на фото Blue
- `prod-violet.jpg` — на фото Violet
- `prod-mokko.jpg` — на фото Mokko

Имена оставьте прежними — сайт всё подхватит автоматически.
